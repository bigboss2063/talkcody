## TalkCody 愿景

构建一套面向开发者的 全流程 AI Coding Agent 平台：
从写代码、测试、文档，到发布、上线、运营，实现端到端自动化开发，让编程变得更快、更便宜、更简单。

## TalkCody 里程碑1：Top AI Coding Agent

<Callout type="success">
目标： 最低成本+最快速度+最高正确率 生成可运行代码
</Callout>

### 1.1 速度

#### 1.1.1 多 Task 并行

一个 Project内，一个Task开发功能，一个Task 修复 bug，一个Task写文档，多个Task并行工作。

- [x] 支持多个 task 并行执行
- [x] 和 Git Worktree 集成
- [x] 支持 task 后台执行

#### 1.1.2 多 Agent 协作并行

实现 Orchestrator-Worker 模式：主 Agent 拆解任务，Coder、Tester、Writer Agent 并行执行。

#### 1.1.3 支持 1到2小时的 long run tasks

### 1.2 成本

- [x] 支持 Claude 订阅

- [ ] 支持 Hook 机制
